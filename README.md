# jaffamonkey.com

This site is built with [gatsby](https://www.gatsbyjs.org/).

### Development

Install dependencies

```
yarn
```

Run local server on http://localhost:8000

```
npm start
```

Build files

```
npm run build
```

### Deployment

```
npm run s3:upload
```

Revert to last deploy

```
npm run s3-revert
```
